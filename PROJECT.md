# Project Intent

This repository is currently a template project for the Python Agent Framework (PAF).

Its purpose is to provide a reusable baseline structure, conventions, and examples for building module-based, message-driven Python applications.

## Template Status

At this stage, the repository does not represent a single production product intent beyond serving as a template.

## Forking Guidance

When you fork this template for a real project, update this file first with:

- The project's mission and problem domain
- Primary user/stakeholder goals
- Scope boundaries and non-goals
- Key architectural intent and constraints
- Any domain terminology the agent should understand

## Agent Context Guidance

Agents and prompts should use this file as the top-level source of project purpose and intent.
If there is a conflict between generic template wording and fork-specific intent, the fork-specific intent in this file should take precedence.
